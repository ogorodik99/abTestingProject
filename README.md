# AB Testing CTR Analyzer

Проект предназначен для анализа результатов A/B-теста по CTR
(`click-through rate`). Приложение загружает CSV-файл с данными о пользователях,
группах эксперимента, посадочных страницах и конверсиях, очищает данные,
считает CTR по группам и строит простые графики для сравнения результата.

## Что умеет проект

- Загружать A/B-данные из CSV-файла.
- Использовать встроенный файл `Data/ab_data.csv` для быстрого запуска анализа.
- Очищать данные перед расчетом:
  - удалять дубли по `user_id`;
  - приводить `converted` к числовому виду;
  - исключать строки, где группа не соответствует странице
    (`control` -> `old_page`, `treatment` -> `new_page`).
- Считать показы, клики и CTR для каждой группы.
- Показывать, у какой группы CTR выше, и разницу в процентных пунктах.
- Сохранять текстовый отчет в `Output/ctr_report.txt`.
- Сохранять график CTR в `Graphics/ctr_graph.png`.
- Открывать графический интерфейс на Tkinter для запуска анализа,
  просмотра результата и редактирования `ab_data`.

## Формат входных данных

CSV-файл должен содержать колонки:

```csv
user_id,timestamp,group,landing_page,converted
```

Где:

- `user_id` - идентификатор пользователя;
- `timestamp` - время события;
- `group` - группа A/B-теста, например `control` или `treatment`;
- `landing_page` - показанная страница, например `old_page` или `new_page`;
- `converted` - факт клика/конверсии: `0` или `1`.

## Запуск

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите приложение:

```bash
python main.py
```

После запуска можно:

- нажать **Посчитать CTR**, чтобы проанализировать файл `Data/ab_data.csv`;
- нажать **Загрузить CSV**, чтобы выбрать другой CSV-файл;
- открыть **Графики**, чтобы посмотреть визуализацию CTR;
- открыть **Редактировать ab_data**, чтобы изменить встроенный набор данных.

## Структура проекта

```text
Data/                  исходные данные
Graphics/              сохраненные графики
Output/                текстовые отчеты и служебные файлы
Library/utils.py       общие функции для конфигурации, pickle и отчетов
Scripts/analysis.py    очистка данных и расчет CTR
Scripts/data_manager.py загрузка, сохранение и нормализация данных
Scripts/gui.py         графический интерфейс Tkinter
Scripts/reports.py     создание отчетов и графиков Matplotlib
Scripts/config.py      загрузка настроек из config.ini
main.py                точка входа в приложение
```

## Результат анализа

В результате работы приложение выводит:

- количество строк в исходном файле;
- число удаленных дублей `user_id`;
- число удаленных строк с ошибочным соответствием группы и страницы;
- количество строк, попавших в расчет;
- CTR по каждой группе;
- группу с лучшим CTR и разницу с другой группой.
